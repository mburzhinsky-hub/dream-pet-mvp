# Промты для дальнейшего создания платформы Dream Pet

## 1. Общий промт для разработчика/AI-кодера

Создай MVP платформы Dream Pet — маркетплейс щенков от питомников и заводчиков. Главный принцип: интерфейс должен быть максимально простым, интуитивным и красивым как для покупателей, так и для питомников. Используй логотип Dream Pet из проекта. Нужно сделать адаптивный сайт и мобильное веб-приложение.

Функции покупателя: каталог щенков, фильтрация по породе, городу, полу, цене, возрасту, документам и статусу, карточка щенка, карточка питомника, быстрая связь через телефон/Telegram/WhatsApp.

Функции питомника: личный кабинет, добавление помёта по единому шаблону, добавление каждого щенка отдельной карточкой, загрузка фото, указание пола, веса, возраста, окраса, темперамента, документов, статуса брони, контактов.

Функции платформы: ручная модерация объявлений, отметка “проверенный питомник”, платное размещение помёта, поднятие объявления, рекламный баннер сверху.

Стиль: чистый премиальный интерфейс, белый фон, золотой акцент из логотипа, мягкие карточки, крупные фото щенков, минимум лишнего текста, простые CTA-кнопки.

## 2. Промт для UX/UI

Спроектируй UX/UI для платформы Dream Pet. Это сервис поиска щенков от питомников. Пользователь должен за 3 действия понять, как найти щенка: выбрать породу/город, посмотреть карточки, связаться с питомником. Заводчик должен за 3–5 минут создать объявление помёта по единому шаблону.

Сделай дизайн премиальным, тёплым и доверительным. Используй золотой цвет логотипа, светлый фон, мягкие скругления, крупные фотографии и понятные фильтры. Избегай перегруженности. Основная эмоция: “здесь безопасно и удобно выбрать щенка”.

Нужные экраны: главная, каталог, карточка щенка, карточка питомника, форма добавления объявления, мобильная версия.

## 3. Промт для backend

Создай backend для Dream Pet на Node.js/NestJS или Express + PostgreSQL. Нужны сущности: User, Kennel, Litter, Puppy, Breed, City, Favorite, Lead, Payment, Banner. Добавь авторизацию питомников, роли buyer/breeder/admin, модерацию объявлений, загрузку изображений, статусы puppy: available/reserved/sold, статусы объявления: draft/pending/approved/rejected.

API должно поддерживать: регистрацию питомника, создание питомника, создание помёта, добавление щенков, фильтрацию каталога, избранное, отправку заявки, админ-модерацию, оплату размещения объявления.

## 4. Промт для SEO

Подготовь SEO-структуру для Dream Pet. Нужны посадочные страницы под породы и города: “купить щенка корги в Москве”, “щенки шпицев в Санкт-Петербурге”, “питомники лабрадоров”. Для каждой страницы создай title, description, H1, блок FAQ, текст 1500–2500 знаков без переспама и структуру внутренней перелинковки.

Цель: получать органический трафик из поиска по породам, городам и намерению купить щенка.

## 5. Промт для монетизации

Разработай монетизацию Dream Pet. Базовая идея: платное размещение помёта, премиум-продвижение, подписки для питомников, рекламные баннеры от брендов кормов, ветклиник, груминга и зоосервисов. Предложи тарифы для MVP, unit-экономику, метрики и модель роста.
