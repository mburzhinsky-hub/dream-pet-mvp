# Dream Pet MVP

MVP платформы для поиска щенков от питомников и заводчиков. Сделан как красивый адаптивный сайт/мобильное веб-приложение на React + Vite.

## Что внутри

- Каталог щенков с фильтрами по породе, городу, полу, цене, статусу и темпераменту.
- Карточка каждого щенка: фото, пол, вес, возраст, темперамент, документы, статус.
- Блок питомников с единым шаблоном подачи.
- Кабинет заводчика: форма добавления объявления помёта/щенка.
- Демо-данные и сохранение новых объявлений в `localStorage`.
- Логотип Dream Pet в `public/logo.jpg`.
- Адаптивный интерфейс под телефон.

## Запуск локально

```bash
npm install
npm run dev
```

Откройте адрес, который покажет Vite, обычно `http://localhost:5173`.

## Сборка

```bash
npm run build
```

Готовая сборка будет в папке `dist`.

## Загрузка на GitHub

git init
